ABSTRACT
This contribution proposes a fundamentally new network architecture for IMT-2030 (6G) and beyond, designated Velsanet. Unlike incremental extensions of existing Internet infrastructure, Velsanet introduces a Topology-First, Packet-Free, and Optically-Native design. Network space is defined prior to addressing, and E2E connectivity is established through physically isolated parallel optical cores. Grounded in five polyhedral primitives (T₄, C₆, O₈, D₁₂, I₂₀), a three-layer AI structure (PAI / AAI / AsAI), and a projection-based global governance model (Q2–Q7).
PROPOSED WORK ITEMS FOR SG13
WORK ITEM A
Topology-First Architecture
Formal definition of pre-coordinate topological space · Study of polyhedral structural primitives as network space grammar
WORK ITEM B
Optical-Native E2E Connectivity
Study of physically isolated multi-core optical path architecture · Evaluation of MOCT hardware for deterministic parallel connectivity
WORK ITEM C
Structurally Embedded AI

WORK ITEM D
Projection-Based Sovereign Governance
