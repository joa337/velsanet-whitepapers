# Papers

This directory contains personal research papers written as part of the broader Velsanet architecture study.

These documents are independent conceptual or structural papers prepared for private research, archival organization, and idea development.

## Notice

The papers in this directory are:

- personally written research documents
- not submitted to any academic journal or conference
- not peer-reviewed publications
- not official standards contributions unless explicitly stated otherwise

They are maintained as research records and conceptual papers exploring structural, topological, AI-native, and network-architecture ideas related to Velsanet.

## Purpose

This directory exists to organize paper-style documents that are more academic in format than the white papers in this repository.

While the `whitepapers` directory focuses on broad architectural explanation, this `papers` directory is intended for:

- conceptual papers
- structural research notes
- architecture-oriented essays
- topology and network theory papers

## Scope

Topics in this directory may include:

- Structure Switching
- topology-defined communication
- rhizomatic and polyhedral network models
- hypercube-based network structures
- AI-native network architecture
- related theoretical extensions of Velsanet

## Note

These papers should be understood as personal research work and idea records, not as formally published academic output.
