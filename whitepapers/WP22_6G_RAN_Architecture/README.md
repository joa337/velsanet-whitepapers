WP22 — Velsanet H6-Based 6G RAN Architecture
Structure-Native Radio Access Without DU/CU
Version 1.0 | 2026-03-07

This white paper defines how a 6G ultra-small cell base station connects to one face of the H6 node via MOCT ribbon cable (5m+), and why this structure eliminates the need for DU and CU.
Mobility management is defined as a system: O8 as decision authority, V-MCU as executor, device Identity as persistent anchor across all movement.
