# Velsanet Whitepapers

This directory contains technical whitepapers published by the Velsanet Architecture Group.

All documents are released in full under the Velsanet open-specification policy. Compliance and interoperability are governed by certification against the register maps and interface specifications defined in each paper.

---

## 📄 Whitepapers

### WP#24: Cognition Cube Physical Specification (HBM / HBF)
- **File**: `WP24_Cognition_Cube_Physical_Spec.docx`
- **Version**: v1.1 (2026)
- **Summary**: WORM register map, HBF ACL bitfields, Transform Engine registers, electrical interfaces, and CH11 functional safety design.

---

## 🔍 Verification Principle

- **Standardized**: Register semantics, lock behavior, ACL encoding, audit-log format, safety state transitions.
- **Out of scope**: Specific silicon processes, vendor IP blocks, connector part numbers (implementation-layer choices).
- **Compliance**: Certified against the register map defined in each specification.
