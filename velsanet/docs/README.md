# Velsanet Documentation

This directory contains all documentation for the Velsanet project.

---

## Directory Structure

| Folder | Purpose |
|--------|---------|
| `itu-t/` | Official ITU‑T submissions — **read‑only** archive |
| `specs/` | Active working specifications — **editable** design documents |
| `references/` *(optional)* | External references, white papers, related standards |

---

## Workflow

1. **Specs first** — all design changes happen in `specs/`
2. **Test against implementation** — validate changes in `../src/`
3. **Freeze for ITU‑T** — when ready, copy the final version to `itu-t/`
4. **Archive** — keep a version history in `itu-t/` for each formal submission
